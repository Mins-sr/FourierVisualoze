# Fourier Visualoze

3Blue1Brown-style Fourier visualization site for exploring frequency components by winding a time-domain signal around a circle.

## Features

- Composite waveform controls with editable frequency, amplitude, and phase
- Winding-plane visualization with center of mass
- Frequency sweep spectrum with peak labels
- Presets for pure tone, 2Hz + 3Hz, beats, and square-wave-like signals
- Responsive UI for desktop and mobile

## Development

```bash
npm install
npm run dev
```
